# Alpha.date AI — Помощник

Расширение для удобной работы на сайте знакомств alpha.date.

## Что включает в себя расширение

- Умный AI-помощник для генерации сообщений
- Игровой автомат (развлекательная функция)
- Дополнительные инструменты для комфортного использования сайта

## Установка

1. Скачай архив с последней версией
2. Распакуй его в любую папку
3. Открой Chrome и перейди по адресу `chrome://extensions/`
4. Включи **"Режим разработчика"** (переключатель в правом верхнем углу)
5. Нажми кнопку **"Загрузить распакованное расширение"**
6. Выбери папку, куда распаковал архив

## Важная информация

- Расширение **не из Chrome Web Store**
- При установке браузер может показывать стандартные предупреждения
- Автор не несёт ответственности за возможные последствия использования
